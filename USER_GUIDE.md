## 🎯 PRÉSENTATION

GestApp est une plateforme moderne de gestion des activités universitaires conçue pour simplifier le suivi, l'organisation et l'analyse des activités académiques et administratives.

### Fonctionnalités Principales
- **Gestion des objectifs** pédagogiques et administratifs
- **Suivi des activités** avec indicateurs de performance
- **Dashboard analytique** avec graphiques interactifs
- **Exports personnalisés** en CSV et JSON
- **Interface responsive** accessible sur tous les appareils

### Rôles Utilisateurs
- **SuperAdmin** : Accès complet à toutes les fonctionnalités
- **Président** : Supervision stratégique et rapports
- **Admin** : Gestion opérationnelle quotidienne
- **Service** : Gestion des activités de service

## 🔐 PREMIÈRE CONNEXION

### Étape 1 : Accéder à la Plateforme
1. Ouvrez votre navigateur web
2. Rendez-vous à l'adresse : `https://gestapp.university.edu`
3. Cliquez sur "Se connecter"

### Étape 2 : S'authentifier
1. **Email** : Entrez votre adresse email universitaire
2. **Mot de passe** : Entrez votre mot de passe
3. **Se souvenir de moi** : Optionnel pour rester connecté
4. Cliquez sur "Connexion"

### Étape 3 : Première Connexion
- Si c'est votre première connexion, vous devrez changer votre mot de passe
- Choisissez un mot de passe sécurisé (8 caractères minimum)
- Confirmez le nouveau mot de passe

### 🔑 Mot de Passe Oublié
1. Cliquez sur "Mot de passe oublié ?"
2. Entrez votre email universitaire
3. Suivez les instructions envoyées par email

## 🎯 GESTION DES OBJECTIFS

### Créer un Objectif
1. **Accédez** à la section "Objectifs" depuis le menu
2. **Cliquez** sur "Nouvel Objectif"
3. **Remplissez** le formulaire :
   - **Libellé** : Nom de l'objectif (ex: "Amélioration de la qualité")
   - **Description** : Détails de l'objectif (optionnel)
4. **Cliquez** sur "Enregistrer"

### Gérer les Sous-Objectifs
1. **Accédez** à "Sous-Objectifs" dans le menu Objectifs
2. **Sélectionnez** l'objectif parent
3. **Créez** des sous-objectifs spécifiques
4. **LieZ** les activités aux sous-objectifs

### Modifier un Objectif
1. **Repérez** l'objectif dans la liste
2. **Cliquez** sur l'icône d'édition (✏️)
3. **Modifiez** les informations nécessaires
4. **Sauvegardez** les changements

### Supprimer un Objectif
⚠️ **Attention** : La suppression est irréversible
1. **Cliquez** sur l'icône de suppression (🗑️)
2. **Confirmez** la suppression dans la fenêtre modale

## 📋 GESTION DES ACTIVITÉS

### Créer une Activité
1. **Accédez** à la section "Activités"
2. **Cliquez** sur "Nouvelle Activité"
3. **Remplissez** le formulaire complet :

#### Informations Principales
- **Objectif Global** : Sélectionnez dans la liste déroulante
- **Sous-Objectif** : Choisissez le sous-objectif pertinent
- **Service** : Sélectionnez le service responsable
- **Période** : Choisissez la période académique

#### Détails de l'Activité
- **Libellé** : Nom clair et descriptif
- **Indicateur** : Métrique de mesure
- **Cible** : Objectif à atteindre
- **Coût** : Budget alloué (en euros)
- **Source de financement** : Origine des fonds
- **Structure** : Structure responsable
- **Commentaires** : Informations additionnelles

4. **Cliquez** sur "Enregistrer l'activité"

### Consulter les Activités
- **Tableau principal** : Liste complète avec pagination
- **Filtres** : Recherche par service, objectif, statut
- **Tri** : Cliquer sur les en-têtes de colonnes
- **Actions** : Voir, modifier, supprimer chaque activité

### Statuts des Activités
- 🟢 **Active** : En cours de réalisation
- 🟡 **En attente** : En attente de validation

### Modifier une Activité
1. **Cliquez** sur l'icône d'édition (✏️)
2. **Modifiez** les champs nécessaires
3. **Cliquez** sur "Mettre à jour"

## 📈 DASHBOARD ANALYTIQUE

### Accès au Dashboard
- **Disponible** pour les rôles Admin et supérieurs
- **Accédez** via "Analytique" dans le menu de navigation

### KPIs en Temps Réel
- **Objectifs Globaux** : Total et taux de sous-objectifs
- **Activités Totales** : Répartition actives/en attente
- **Budget Total** : Somme allouée et moyenne par activité
- **Services** : Nombre et utilisateurs

### Graphiques Interactifs

#### Répartition par Service
- **Type** : Graphique en barres
- **Informations** : Nombre d'activités par service
- **Interaction** : Survolez pour voir les détails

#### Statut des Activités
- **Type** : Graphique circulaire
- **Informations** : Répartition actives/en attente
- **Taux de complétion** : Pourcentage affiché

#### Budget par Service
- **Type** : Graphique en camembert
- **Informations** : Répartition budgétaire
- **Couleurs** : Différenciation visuelle

#### Évolution Mensuelle
- **Type** : Graphique linéaire
- **Période** : 12 derniers mois
- **Tendance** : Visualisation de la croissance

### Tableaux de Performance
- **Top 5 Services** : Les plus actifs avec budgets
- **Top 5 Objectifs** : Les plus performants
- **Activités Récentes** : Dernières créées

### Fonctionnalités du Dashboard
- **Rafraîchissement automatique** : Toutes les 5 minutes
- **Export direct** : Bouton d'export en CSV
- **Impression** : Version imprimable optimisée
- **Responsive** : Adapté mobile et tablette

## 📤 EXPORTS ET RAPPORTS

### Accès aux Exports
1. **Menu Administration** → "Exporter les données"
2. **Ou exports rapides** directement depuis les pages

### Types d'Exports

#### Export d'Activités
- **Format** : CSV (compatible Excel)
- **Filtres disponibles** :
  - Service spécifique
  - Statut (actives/en attente)
  - Période personnalisée
- **Contenu** : Liste complète avec toutes les informations

#### Export de Statistiques Globales
- **Format** : CSV
- **Contenu** :
  - Statistiques des objectifs
  - Statistiques des activités
  - Répartition par service
  - Indicateurs clés

#### Export de Performance
- **Format** : CSV
- **Période** : 6 derniers mois
- **Contenu** :
  - Évolution mensuelle
  - Performance par service
  - Taux de croissance
  - Indicateurs de performance

### Configuration d'Export
1. **Sélectionnez** le type d'export
2. **Choisissez** le format (CSV recommandé)
3. **Appliquez** les filtres souhaités
4. **Aperçu** : Vérifiez avant d'exporter
5. **Téléchargez** : Cliquez sur "Exporter"

### Exports Rapides
- **Activités complètes** : Un clic depuis dashboard
- **Statistiques globales** : Export instantané
- **Rapport performance** : Données des 6 derniers mois

## 📚 GUIDE DES SERVICES

### Gestion des Documents
1. **Accédez** à "Guides" dans le menu
2. **Cliquez** sur "Ajouter un guide"
3. **Sélectionnez** un fichier (PDF, Excel, Word)
4. **Taille maximale** : 5MB
5. **Formats acceptés** : PDF, XLSX, DOCX

### Types de Documents
- **Guides procédures** : Instructions administratives
- **Formulaires** : Modèles à utiliser
- **Rapports** : Documents de référence
- **Politiques** : Directives institutionnelles

### Organisation des Guides
- **Classés par service** : Facile à retrouver
- **Recherche** : Fonction de recherche intégrée
- **Téléchargement** : Accès direct aux fichiers

## 🔧 GESTION DES PÉRIODES

### Créer une Période
1. **Accédez** à "Périodes" dans le menu
2. **Cliquez** sur "Nouvelle période"
3. **Définissez** :
   - **Libellé** : Ex: "Semestre 1 2024-2025"
   - **Date de début** : Début de la période
   - **Date de fin** : Fin de la période
4. **Activez** la période si nécessaire

### Périodes Types
- **Semestres** : Semestre 1, Semestre 2
- **Trimestres** : T1, T2, T3, T4
- **Années** : Année académique complète
- **Personnalisées** : Périodes spécifiques
